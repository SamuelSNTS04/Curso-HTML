Estrutura básica do HTML

O HTML é uma linguagem de marcação ultilizada na web. Ela é composta de uma estrutura simplória baseada em três partes: a abertura e fechamento da tag HTML onde o navegador vai assimilar que isso é um HTML , a abertura e o fechamento da tag body onde será exibido tudo ao usuário , e abertura e o fechamento da tag head que carrega o que for necessário antes de exibir o body.

Tags e atributos das tags

O HTML possui uma série de tags, mas o que são essas tags? O corpo do HTML é formado por tags, elas trazem o conceito de melhorar um texto, fazer menções através de links, marcações, abreviações, uma quebra de linha, separação de um título de um subtítulo, por fim toda a organização visual de um HTML é feito por tags.

Algumas dessas tags podem ser acompanhadas de atributos, que fazem um tipo de configuração para essa determinada tag , aparecendo de uma forma diferente no navegador.

Listas ordenadas e não ordenadas

As listas, conceitualmente traz uma referência clara do seu objetivo que é ordenar algum tipo de dado que deve ser mostrado ao usuário. Existem listas ordenadas e não ordenadas, a grande diferença é que as ordenadas vem com um conjunto alfanumérico especificando em qual item está, exemplo de uma lista ordenada é um sumário que indica em qual página do livro você encontra determinado tema. A tag para fazer uma lista ordenada é a tag "ol" seguida de quantos items desejar listar com a tag "li".

Já as não ordenadas, trabalha dentro do mesmo conceito mas quando se trata da indicação não é feita por um número mas sim por um ícone, seja ele um quadrado, uma esfera, um triângulo, etc. A tag para fazer uma lista não ordenada é a tag "ul" seguida de quantos items desejar listar com a tag "li".

Links

O conceito de link vem de navegação. É através de um ou mais links que podemos navegar de um site para outro ou mesmo , navegar dentro de um mesmo site com tópicos específicos. A tag para fazer um link é a tag "a"
